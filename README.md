# Manhattan-Distance-Kata-TDD

Repositorio de práctica de **Test-Driven Development (TDD)**.

## Requisitos

Antes de comenzar, se debe de tener instalado lo siguiente:

- [Python](https://www.python.org/downloads/)
- [Pytest](https://docs.pytest.org/en/stable/getting-started.html)

## Instrucciones de Uso

Siga estos pasos para ejecutar las pruebas y practicar TDD:

1. Clonar el repositorio en su máquina local:
    ```bash
    git clone https://github.com/SamEsteb/Manhattan-Distance-Kata-TDD/
    cd Manhattan-Distance-Kata-TDD
    ```

2. Instala las dependencias necesarias utilizando el archivo `requirements.txt`:
    ```bash
    pip install -r requirements.txt
    ``` 

3. Ejecuta las pruebas con `pytest` desde el directorio raíz del proyecto:
    ```bash
    pytest
    ```

4. Para revisar los cambios paso a paso, se pueden inspeccionar los commits del repositorio:
    ```bash
    git log --oneline
    git checkout <ID_DEL_COMMIT>
    ```
